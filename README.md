# skipschool

湖南文理学院自动刷课插件(已测试)

功能：

 - 文理学院成考课程 自动刷课
 - 支持多线程 10倍速 刷课
 - 下载解压后进入谷歌浏览器扩展程序点击加载解压扩展即可自动启用
 - 可以微信搜索 [李大炮工作室] 关注获取后续消息
